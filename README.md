# Master Collection App

Webflow Designer Extension for installing purchased Master Collection packages into the buyer's current Webflow site/page.

This folder maps to:

```text
https://github.com/Imacx-maria/master-collection-app.git
```

## Boundary

The app owns:

- current Webflow site/page detection
- font checklist inside Webflow
- asset upload into the buyer's Webflow site
- XscpData patching
- clipboard paste/install handoff

The app does not own catalog, checkout, account library, purchase entitlement, or install-code creation. Those belong to the website/platform.
